Version Compatibility:
</H2>

Houdini 18.0+

<H1>

</H1>

huskStandaloneSubmitter.py 
DeadlineRepository \custom\scripts\Submission

HuskStandalone (DIR)
DeadlineRepository \custom\plugins\HuskStandalone
